# -DSBDA-MiniProject
 Sales Prediction using Regression and Data Visualization using Power BI
 #  Sales Prediction Project

##  Project Overview
This project focuses on predicting sales using Machine Learning and visualizing insights using Power BI. A Linear Regression model is used to predict sales based on multiple factors.

---
##  Objectives
- Perform data preprocessing (cleaning, handling missing values, outliers)
- Build a regression model for prediction
- Achieve R² Score ≥ 0.75
- Create visualizations using Python
- Develop an interactive dashboard using Power BI

---
## Dataset
The dataset contains:
- Month
- Year
- Advertising_Spend
- Discount
- Sales (Target Variable)

---
## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Microsoft Power BI

---
## Model Used
- Linear Regression

---
## Visualizations
- Line Chart (Monthly Trend)
- Bar Chart (Year-wise Sales)
- Pie Chart (Sales Distribution)
- Heatmap (Feature Correlation)

---
## Power BI Dashboard
Includes:
- KPI Cards (Total, Avg, Max Sales)
- Line Chart (Trend)
- Bar Chart (Comparison)
- Slicer (Year Filter)

---
## Model Performance
- R² Score: ≥ 0.75
- Good prediction accuracy achieved
